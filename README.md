# CodeAlpha Chatbot 🤖

## 📌 Internship Project
This repository contains **Task 4: Basic Chatbot** completed as part of my **Python Internship at CodeAlpha**.  
The project demonstrates Python concepts such as loops, conditionals, string operations, and user interaction.

---

## 📝 Project Description
The Basic Chatbot is a simple text-based program:
- Responds to greetings like "hi", "hello", or "hey".
- Answers basic questions such as "what is your name" or "how are you".
- Provides help instructions.
- Shows current **time** and **date**.
- Exits gracefully when the user types "bye".

---

## ⚙️ Technologies Used
- Python 3
- Loops & conditionals
- String operations
- Datetime module
